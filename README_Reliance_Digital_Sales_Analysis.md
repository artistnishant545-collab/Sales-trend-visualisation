# 📊 Reliance Digital Sales Trend Analysis

## 📌 Project Overview
This project focuses on analyzing and visualizing sales data of Reliance Digital using Python. The objective is to identify sales trends, category performance, seasonal patterns, and overall business growth through data analytics and visualization techniques.

## 🎯 Objectives
- Analyze monthly sales performance
- Identify top-performing product categories
- Evaluate yearly revenue growth
- Understand seasonal sales trends
- Visualize business insights using charts and dashboards

## 🗂 Dataset Information
Columns:
- Month
- Category
- Units_Sold
- Revenue_INR

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- GitHub

## 📈 Visualizations Included
- Monthly Revenue Trend
- Monthly Units Sold Trend
- Category-wise Revenue Analysis
- Revenue Share Pie Chart
- Yearly Growth Analysis
- Revenue Heatmap
- Moving Average Trend
- Correlation Analysis

## 🔍 Key Findings
- Revenue shows consistent growth from 2023–2025.
- Festive season months generate peak sales.
- Smartphones contribute the highest revenue share.
- Positive year-over-year business growth.

## 🚀 How to Run
```bash
pip install -r requirements.txt
python main.py
```
